Point Size Variants
      When typography still meant cutting letters into individual blocks of metal there was one advantage: letterforms would have a natural and slight variation at different point-sizes. Thin-stroke width could remain consistent as the point size increased. When typefaces became digitized designers would have to compromise on a single relative stroke size. The contrast would end up too great at smaller sizes leading to a “dazzle” which would render the text difficult to read. The contrasting stroke thickness would end-up too small at larger sizes, leading to a loss in the feeling of elegance or refinement.
      indestructible type* aims to combat this by introducing point size variants. The goal is to have a separate font-pack optimized for 6 point, 12 point, 24 point, 48 point and 96 point sizes. This would allow the designer to pick the font best suited for the size being used. At this time Bodoni* only goes up to 12 points.

Smart Typography

Kerning Pairs
      Bodoni* has been kerned to make sure all letter combos look evenly spaced, not just a few. This means that along with the common couplings like “Tr” and “Ye”, uncommon ones such as “yA” look evenly spaced.

Ligatures
      Some letter combinations create conflict. For example, the upper curve of the “f” overlaps with the dot of the “i” To combat this Bodoni* includes a complete set of ligatures. Ligatures are separate glyphs (letters) that automatically replace conflicting letter combos.
