UMBRA MEANING SHADOW

To my knowledge, Umbra* is the first ever font to respond to the actual time of the user. The shadow of Umbra* changes location throughout the day in relation to the time. Come back and refresh this webpage every couple of hours, and watch as like magic, the shadow moves to the sun!

OPENTYPE VARIABLE FONTS

Umbra* is built using OpenType variable font technology, a new font standard that allows for user-controlled variables. In the past, font metrics, such as boldness or oblique angle, were limited to the few predetermined options made by the font designer. Users wanting precise control over metrics such as boldness were confined to using digital trickery or the few fonts with a large enough family to support their needs. With the introduction of OpenType variable fonts, users will be able to control font metrics using sliders, allowing endless possibilities.

A LIMITLESS TECHNOLOGY

OpenType variable fonts not only alow the creation of user control inputs for familiar variables, such as font weight or width, but enable the designer to create new and unheard of variables. Many new and exciting experements are being done with this technology, see Axis-Praxis for some examples. What's most exciting is all of the amazing things that it will alow future makers and thinkers to do. Things like a font that responds to the actual time of the user. Things like a font with user-controled wind. Things not yet conceived of.
