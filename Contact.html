Frequently Asked Questions

What’s the difference between a commercial and personal license?
If it's in any way used for commerce, than it’s commercial and should use the commercial license. This includes logos, professional websites, t-shirts, etc. If you aren't making money off of it, if nobody outside of yourself sees it, it's personal.


I bought a version of a font here, and now I want the newer version, how do I get it without paying again?
All font downloads should come with the option to subscribe to email updates of the font. In addition, all downloaded .zip files should include README.txt file with instruction on downloading the latest version. If both of those fail shoot me an email?


I want to update my version of a font, but my computer says that it’s already installed. How do I fix this?
You just have to uninstall it and then install it again. If you are using Linux, check under ~/.local/share/fonts directory. If you are in Windows, there should be a fonts file under Appearance and Personalization, which is in the control panel. If you are using a Mac, you should install Linux.
